Программа для работы с .img .dff .txd файлами
Функционал:

- .img Batch

	* Позволяет батчем создать рендеры всех .dff моделей в .img файле (либо уже распакованные из папки)
 Source Type
     Если установлен чекбокс 'Batch from IMG', то можем выбрать .img файл для дальнейшей работы с ним.
     Если установлен чекбокс 'Batch from Folder', то вместо .img файла выбираем папку, куда был заранее распакован .img, либо где лежат нужные Вам .dff модели и .txd текстуры

 Output Image Size
     Вводим вертикальный размер изображения для каждого рендера (по умолчанию 500px), горизонтальный размер рассчитается автоматически
 Textures
     При активном чекбоксе Auto-textures будет использоваться моя экспериментальная функция подбора текстур к объектам (хорошо подходит для моделей одежды, у которых не указан материал, например).
      В случаях, если на 1 модель найдётся несколько текстур - будет отрендерен вариант с каждой текстурой, которая по мнению алгоритма подходит к модели.
	*Функция экспериментальная, нестабильная, может иногда подбирать лишние текстуры, либо не находить вообще.
	*Ищет сначала по процентам наибольших совпадений материалов, далее по процентам совпадений имён, имеет определения типов некоторых моделей.

     При активном чекбоксе No textures к моделям не будут применены никакие текстуры, они будут в сером свете, со слабым освещением для выделения полигонов.

При выборе | файла/папки |, в нижней части экрана отобразятся все .dff файлы внутри | .img/выбранной папки |
Можно выбрать необходимые для рендера, либо нажать Select All, чтобы выбрать все
После чего жмём 'Batch Render', пойдёт процесс рендера всех изображений. Прогресс в реальном времени отображается в нижней части окна.

- .img unpack
	*позволяет распаковывать, запаковывать, и просто редактировать .img файлы

    Unpack IMG
        Выбираем нужный .img, выбираем папку, куда хотим распаковать (!!!автоматически создаст подпапку в выбранном месте!!!)

    Pack to IMG
	Выбираем папку, в которой лежит всё содержимое, которое хотим упаковать в .img, далее выбираем место, где сохранить готовый .img

    Edit IMG
	Выбираем нужный .img, открывается отдельное окно со всем его содержимым, можем заменять .dff и .txd файлы на те которые нам нужны, удалять, добавлять новые. (не забывайте делать бэкапы!)
	 При нажатии на save - сохраняет отредактированный файл. Cancel отменяет изменения соответственно

- Viewer
 	*Удобный .img viewer для просмотра .dff моделей

           Чекбоксы:
		Auto-search textures - использует тот же умный алгоритм поиска текстур, что и в .img Batch
		Manual texture selection - позволяет вручную указать желаемый .txd файл перед выбором модели
		No textures (gray model) - модель будет загружена без подбора текстур
		 ***Текстуру можно добавить вручную во время просмотра модели

	Выбираем модель с нужным параметром текстуры, попадаем во Viewer
	Там открывается 4 окна (Основное с моделью, и 3 отдельных окна с элементами HUD, при желании можно закрыть)
	В окне Textures Вы можете включать и отключать нужные текстуры, также можете загрузить ещё текстуры кнопкой "Choose More..."
	В окне Geometry Можно скрывать и показывать parts модели (у транспорта по умолчанию отключен LOD), если модель определена, как одежда, то она будет разъезжаться ан 3 части (3 телосложения)
	В окне HUD отображается текущее положение модели, зум камеры, текущий FPS

Программа может быть нестабильной, она практически не протестирована.
